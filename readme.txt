Optimizes WordPress database with advanced queries, custom tables, UI options, and safe migrations.

== Description ==
Advanced features: Clean revisions/transients/spam, custom cron intervals, UI checkboxes, logs, and email notifications.

== Installation ==
1. Upload to /wp-content/plugins/.
2. Activate in Plugins menu.
3. Configure in Tools > DB Optimizer.
